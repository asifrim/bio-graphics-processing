NAME

gff2png.rb

USAGE

 ./gff2png.rb [ -h | --help ]
              -l | --length length_of_sequence
              [ -i | --infile | < ] your_gff_file.txt
              -o | --outfile your_output.png

DESCRIPTION

This script parses a GFF file and creates a picture using Bio::Graphics. The flavour of GFF required is GFF3 (see www.sequenceontology.org/gff3.shtml).

OPTIONS

 -h,--help::                 Show help
 -l,--length=LENGTH::        Length of sequence. Required.
 -i,--infile=INFILE::        Name of input file. STDIN if not defined.
 -o,--outfile=OUTFILE::      Name of output file. Required.

FORMAT INPUT

GFF3. See www.sequenceontology.org/gff3.shtml

FORMAT OUTPUT

A PNG picture.
